### Run Locally

Clone the project

```bash
  git clone https://github.com/dastageer-hc/order-summary-card
```

Go to the project directory

```bash
  cd order-summary-card
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
